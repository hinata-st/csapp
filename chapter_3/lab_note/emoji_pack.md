# Markdown 表情包（可直接复制）

## 常用状态
- 😀 😄 😎 🤓 🙂
- ✅ ❌ ⚠️ 🚧 🛠️
- 🔥 💡 📌 🧠 🎯

## 学习/做题
- 📚 ✍️ 🧪 📖 🗂️
- 😵 😤 🤯 😌 🥳
- 这一题我会了：😎✅
- 这题有坑：⚠️🕳️
- 终于调通了：🔥🥳

## 调试现场
- 代码跑飞了：🚀💥
- 发现 bug：🐛🔍
- 修复成功：🛠️✅
- 先打印看看：🖨️👀
- 回归测试通过：🧪✅

## 进度标记
- 待办：📝
- 进行中：⏳
- 已完成：✅
- 延后：⏸️
- 阻塞：🚫

## 心情专区
- 今天状态很好：🌞😄
- 脑子有点转不动：🌧️😵
- 快要崩溃：😫💫
- 继续坚持：💪🔥
- 收工：🍵😌

## GitHub 短代码（可选）
> 在 GitHub 风格 Markdown 里，也可以用 `:smile:` 这种写法。

- :smile: :sweat_smile: :thinking: :rocket: :bug: :fire: :white_check_mark: :x: :warning:

## 一行版（评论区常用）
`进度：⏳ | 难度：🤯 | 状态：🛠️ | 结果：✅`

## 组合模板
- 今日任务：📌 [任务名]，当前进度：⏳，预计完成：🕘
- 问题记录：🐛 [现象]，定位中：🔍，修复后：🛠️✅
- 学习总结：📚 [主题]，收获：💡，下一步：🎯
